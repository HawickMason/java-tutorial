# 运算符总结
[[toc]]

以下快速参考总结了 Java 编程语言支持的运算符

## 简单赋值

* = Simple assignment operator

## 算术运算符

* `+`  加            
* `-`  减    
* `*`  乘    
* `/`  除    
* `%`  剩余(取模)

## 一元运算符

* `+`  正数
* `-`  负数
* `++` 递增 1
* `--` 递减 1
* `!`  逻辑互补，反转布尔值的值

## 平等 运算符

* `==`      等于
* `!=`      不等于
* `>`       大于
* `>=`      大于等于
* `<`       小于
* `<=`      小于等于

## 条件

* `&&`      短路 AND
* `||`      短路 OR
* `?:`      三元运算符（是 if-then-else 的简写）

## 类型比较运算符

* instanceof  比较一个对象是否是指定类型

## 按位和位移操作符

* `~`       一元位补
* `<<`      左移
* `>>`      右移
* `>>>`     无符号右移
* `&`       AND
* `^`       按位异或
* `|`       按位或
