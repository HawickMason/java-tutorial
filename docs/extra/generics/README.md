# 泛型
J2SE 5.0 中引入了这种期待已久的增强类型系统的功能，它允许 **类型** 或 **方法** 对各种类型的对象进行操作，同时提供了编译时类型的安全性。它为集合框架增加了编译时类型的安全性，并消除了繁重的工作。
